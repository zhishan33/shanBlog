## 正则表达式

### [A-z]
> 方括号范围字符查找 **[A-z]**不仅能够匹配a-z和A-Z还会匹配[],要匹配字母大小写规则**[a-zA-Z]**


```
	<script>
		var str = "[]";
		var result = /^[A-z]$/.test(str) // true;
		var result2 = /^[a-zA-Z]$/.test(str) // false;
	</script>
```
