## ubuntu svn 客户端

1. 自带source源里面查找rabbitvcs信息

```shell
  sudo apt search rabbitvcs
```
2. 安装rabbitvcs

```shell
sudo apt install rabbitvcs-cli rabbitvcs-core rabbitvcs-gedit rabbitvcs-nautilus
```
