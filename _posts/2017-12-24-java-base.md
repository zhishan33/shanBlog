# static 修饰符

> static 用来修饰共享的成员变量，随着类的加载而运行。

```
  class demo {
    public static String = "static";
  }
  

```



# 代码块

> static 修饰的代码块主要用来初始化类，随着类的加载而运行，类中非静态代码块主要用来初始化对象，与构造函数不同的是，非静态代码块作用与所有的对象，而构造函数只作用与特定的对象。

```
  class demo {
    
    static {
    
    }
    
    {
    
    }
  }
   
    

```
